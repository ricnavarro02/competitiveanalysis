# WWDC25 Brand Guidelines

Apply Apple's WWDC25 visual design system to presentations.

## Quick Reference

### Colors (Primary)

| Role | Hex |
|------|-----|
| Background | `#F5F5F7` |
| Primary Text | `#222426` |
| Highlight Blue | `#7CA9EE` |

### Accent Colors

Green `#8FCB9A` · Yellow `#F5CF8B` · Orange `#F0B582` · Red `#EC938E` · Purple `#9596D4` · Blue `#7CA9EE`

### Typography

- **Titles**: SF Hello Bold
- **Body**: SF Hello Regular (18-30pt, sentence case)
- **Code**: SF Mono Medium (48pt default)
- **Theme default**: SF Pro Semi Expanded Medium

### Layout

- Canvas: 1920×1080pt (outputs to 4K)
- Split 50/50: 820pt content width
- Split 2/3: 1140pt content width
- Corner radius: 8pt (inner), 28pt (outer)

## Usage

When creating WWDC-style presentations:

1. Use the background color `#F5F5F7` for standard slides
2. Apply SF Hello font family for all non-code text
3. Use SF Mono for any code snippets
4. Limit to 3 consecutive alpha/split screen slides
5. Use approved animations only (Fade and Move, Magic Move, Line Draw, Dissolve)
6. Avoid "Special Effects" and "Flip, Spin and Scale" animations
