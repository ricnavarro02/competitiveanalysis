## Industry Best Practices (Research Findings)

### Audience-Driven Formatting
The #1 industry standard: **tailor the format to the audience**. Executives need revenue/strategic impact insights in 1-2 pages. Operational teams need tactical detail. Never serve both from the same document.

### Executive-Level Best Practices
- Lead with **"so what"** — strategic implications, not raw data
- Limit to **2-5 key insights** with recommended actions
- Use **visual comparisons** (positioning maps, comparison matrices)
- Include **threat/opportunity framing**, not encyclopedic competitor profiles
- Each recommendation should be **SMART** (Specific, Measurable, Achievable, Relevant, Time-bound)
- Separate **immediate actions (30 days)** from **strategic investments (6-12 months)**

### Common Frameworks
- **Battlecards**: 1-page competitor snapshots for sales/team use
- **SWOT Analysis**: Classic strategic framework for exec audiences
- **Positioning Maps**: Visual quadrant charts showing competitive landscape
- **Tiered Deliverables**: Different docs for different audiences from same research

### Key Principle
> "Competitive analysis is not a one-and-done project; it's an ongoing discipline."
> Deliverables should be **living documents** that are updated, not static reports.

### Version 3: "Competitive Battlecards" (Per-Competitor Quick Reference)

**What it is:** A set of 1-page battlecards — one per competitor program — designed for quick reference during planning, stakeholder conversations, or program design discussions.

**Structure (per card):**
- **Program Name & Company**
- **Program Overview**: 2-3 sentence summary
- **Key Stats**: Member count, regions, year launched
- **What They Do Well**: 3-4 bullets
- **Where They Fall Short**: 2-3 bullets
- **How Apple Compares**: Direct positioning against this specific competitor
- **Key Takeaway**: One strategic insight for Apple

**Programs covered:** Microsoft MVP, AWS Heroes, Google GDE, Salesforce MVP, Twilio Champions (and potentially Docker Captains, Auth0 Ambassadors)

**Why this works:**
- Modular — update one card without touching the whole report
- Highly actionable — grab the relevant card before a specific discussion
- No extraneous information — each card is focused on one competitor
- Industry standard format used by sales teams and product teams globally
- Easy to keep current as competitor programs evolve

**Best for:** Quick reference during meetings, onboarding new team members, stakeholder Q&A prep
